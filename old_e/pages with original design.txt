для того, чтобы по ссылке на материал открывалась страница с оригинальным дизайном необходимо 

1) создать в папке шаблона файл

p_(ID страницы).html - где должен быть оригинальный дизайн страницы, элементы доступные для использования 

PAGE_NAZV, PAGE_FLAG, PAGE_TXT, COMMENTS_NUMBER
$title_head, show_in_bigpic, comments, comm_allow (?)

2) если есть p_(ID страницы)_head.html - то этот файл будет использован вместо head по умолчанию

3) если есть p_(ID страницы)_halfhead.html - то этот файл будет _добавлен_ к head исполз. по умолчанию (или оригинальному)
